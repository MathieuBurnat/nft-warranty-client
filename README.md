# Panels

- Admin Panel

  - Check Admin
  - Add/Remove Admins
  - Add Seller

- Seller Panel

  - Add Product
  - Sell Product
  - Service Product

- Customer Panel

  - Track Warrenty
  - Track Repairs
  - Check if Owner
  - Transfer Ownership
